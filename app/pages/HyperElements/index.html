HyperElements Documentation

Overview

HyperElements are a collection of CustomElements that allow you to declaratively enhance HTML. What if you could send DELETE, or PUT requests from forms and links, and display the status of requests without writing a single line of JavaScript? HyperElements make this possible.

The Elements

<hyper-fetch>
provides a fetch capabilitiy to forms and links
<hyper-link>
enhances <a> elements to send fetch requests
<hyper-form>
enhances <form> elements to send fetch requests
<hyper-status>
displays the loading, success, and error states of fetch requests

Usage

To use HyperElements, you need to include the HyperElements.mjs file in your project. You can do this by adding the following line to your HTML file: 

<script src="https://jessehattabaugh.com/_public/browser/HyperElements.mjs"></script>

Once you have included the HyperElements file, you can use the elements in your HTML. For example, to use the <hyper-link> element, you would write: 

<hyper-link method="DELETE">
	<a href="/books/1?method=DELETE">delete book</a>
</hyper-link>

You may want to download the code and host it yourself.

<hyper-link>

The <hyper-link> element enhances <a> elements to send fetch requests. The method attribute specifies the HTTP method to use. You are expected to nest an <a> element inside that has an href attribute. The <hyper-link> element will intercept the click event and send a fetch request using the method specified in the method attribute. Note that if JavaScript is disabled, clicking the link will perform a GET like normal. So, you may want to include the method in the URL as a query parameter and handle that on the server. 

<hyper-link method="DELETE">
	<a href="/books/1?method=DELETE">delete book</a>
</hyper-link>

<hyper-form>

The <hyper-form> element enhances <form> elements to send fetch requests. The method attribute specifies the HTTP method to use. You are expected to nest a <form> element that has an action attribute. The <hyper-form> element will intercept the submit event and send a fetch request using the method specified in the method attribute. The <form> can also have a method attribute, but it will be ignored. Note that if JavaScript is disabled, submitting the form will perform a GET like normal. So, you may want to use a hidden input and handle that on the server.

example

<hyper-form method="PUT">
	<form action="/books/1" method="PUT">
		<input type="hidden" name="method" value="PUT">
		<input type="text" name="title" value="new title">
		<input type="submit" value="update book">
	</form>
</hyper-form>

<hyper-status>

The <hyper-status> element displays the loading, success, and error states of fetch requests. You must include the for attribute and set it to the id of the element you want to monitor. This can be a comma separated list of ids. To display the status, you can nest elements with the slot attribute set to loading, success, or error. If you don't include these elements, nothing will be displayed.

example

<hyper-link id="deleteAuthor" method="DELETE">
	<a href="/authors/2?method=DELETE">delete author</a>
</hyper-link>
<hyper-status for="deleteAuthor">
	<span slot="loading">deleting author...</span>
	<span slot="success">author deleted!</span>
	<span slot="error">error deleting author!</span>
</hyper-status>

<hyper-fetch>

The <hyper-fetch> element provides a fetch capability to forms and links. It is the base element that the other elements extend. You can use it to create your own custom elements that send fetch requests. The method attribute specifies the HTTP method to use. The id attribute is added to events dispatched by the element.

example

import { HyperFetch } from 'https://jessehattabaugh.com/_public/browser/HyperElements.mjs';

class MyElement extends HyperFetch {
	constructor() {
		super();
		this.method='PUT';
		this.url='http://some.url';
		this.handleClick = this.handleClick.bind(this);
	}
	connectedCallback() {
		this.addEventListener('click', this.handleClick);
	}
	async handleClick() {
		const data = this.fetch(this.url);
		console.log(data);
	}
}
customElements.define('my-element', MyElement);

You can then use the <my-element> element in your HTML like this: 

<my-element>click me</my-element>
