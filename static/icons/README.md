# PWA Icons

This directory contains the Progressive Web App icons for the website.

For a production deployment, you should replace these placeholder files with actual icon files:

- `icon-192x192.png` - 192x192 pixel PNG icon
- `icon-512x512.png` - 512x512 pixel PNG icon

Icons should be square, high contrast, and work well when maskable (with safe areas for different device shapes).

## Icon Guidelines
- Use simple, recognizable designs
- Ensure good contrast for visibility
- Consider how the icon looks at different sizes
- Test the maskable versions on different device shapes
